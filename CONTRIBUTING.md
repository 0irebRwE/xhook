## How to run locally

* Install [Node](http://nodejs.org)

* Install [`grunt-source`](https://github.com/jpillora/grunt-source) and [`serve`](https://npmjs.org/package/serve) with `npm install -g grunt-source serve`

* Fork and clone `git clone https://github.com/MY-USERNAME/xhook`

* Enter project `cd xhook`

* Serve project on port 3000 `serve`

* New Tab `Command+T`

* Watch *src* for changes and will then compile and minify into *dist* with `grunt-source`

* New Tab `Command+T`

* Open Chrome `open http://localhost:3000/`
